Quando digitamos "html:5" no VSCode, ele automaticamente nos mostra um código básico de HTML.
Nesse código, "html lang=" dita o idioma da nossa página. Temos também duas áreas básicas:
A área "<head>" e a área "<body>"
Todo site tem essas duas áreas. A head é a parte de configurações, a body é a parte em que vai aparecer na área branca. 
Tudo que tiver entre os sinais de maior e menor
(por exemplo "<head>") são chamadas de TAGS, nossas marcas.

a tag <meta>, dentro da tag <head> está sendo utilizada 3 vezes para 3 configurações diferentes.
<head>
	<meta charset="UTF-8"> 
# Para configurações de caracteres, pra gente poder utilizar as acentuações do nosso idioma e de alguns outros e etc.
    <meta http-equiv="X-UA-Compatible" content="IE=edge"> #
# sla ele n explicou
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
# Uma melhor configuração de compatibilidade com o Microsoft EDGE.
	<title>Document</title>

Você deve ter percebido que tem também o <title> ali em cima. É nele que modificamos o nome da guia do navegador. Para alterar, você substitui o "Document" por algo que tu queira. como por exemplo "Meu primeiro programa". E como salvar? File > Save. ou CTRL + S.
Quando alteramos o script de alguma forma, na aba do VSCode vai aparecer uma bolotinha, que indica que algo foi alterado recentemente e n foi salvo
Numa matéria do G1 por exemplo, existe o título da matéria, certo?
Para fazer isso, nós vamos no nosso "Olá Mundo" no script, e adicionamos na frente:
"<h1>" e no fim "</h1>", ficando assim: <h1>Olá, Mundo!</h1>
para adicionar o parágrafo, pule uma linha e em seguida "<p>xyz</p>", onde "xyz" é o texto que desejar.

Importante! <h1>/<h1> não transforma a letra maior, transforma em TÍTULO DE NÍVEL 1!

Isso tudo foi a parte de CONTEÚDO! E todo o conteúdo é com a Jornalista, ou seje, o HTML.


Agora, com o Designer, ou melhor dizendo, o CSS:

Há também uma parte de estilo. Isso a gente configura geralmente no <head>, onde a gente coloca uma tag de "<style>XYZ</style>"

	*Dentro do <style>, nós colocamos "body", abrimos chave "{}" e ao digitar "background-color" nós podemos alterar a cor do fundo da página. ex:
<style>
        body {
            background-color: rgb(89, 27, 148); #*
        }
    </style>
	
ATENÇÃO!!!!!!!!!!!!!!!!! TODO FINAL DE MARCAÇÃO EM CSS TEM NO FINAL ";"* !!!!!!
Se por acaso a cor ficar muito forte, deixando o mouse em cima da cor "rgb" abrirá um mini popup onde você vai poder alterar isso.

	*Para alterar a cor do texto, ainda dentro do <style> escrevemos "color: xcor;" ex:

<style>
        body {
            color: lightblue; #Não se preocupe que o VSCode com seus popup vai te ajudar.
        }
    </style>
	
	*Para alterar a fonte, escrevemos:
	<style>
        body {
            font: normal 25pt arial #Onde normal seria ela normal, sem negrito nem nada, "25pt" seria o tamanho da fonte e "arial" a fonte utilizada.
        }
    </style>
vale mencionar que a fonte do título também será alterada, e aumentará ou diminuirá proporcionalmente aos parágrafo abaixo do título.

Todos esses códigos juntos ficarão da seguinte forma:

<style>
        body {
            background-color: rgb(89, 27, 148);
            color: lightblue;
            font: normal 20pt arial;
        }
    </style>

Falta agora o Entregador/Programador, o Javascript.

Geralmente a gente coloca antes de fechar o "<body>" a tag "<script>XYZ</script".
E dentro dele programaremos em Javascript. 

Nota:

Você NÃO diz que vai programar em HTML, você NÃO diz que vai programar em CSS, já que AMBAS NÃO SÃO LINGUAGENS DE PROGRAMAÇÃO(!), diferentemente do Javascript.
HTML é uma Linguagem de MARCAÇÃO/CONTEÚDO!
CSS é uma Linguagem de ESTILO!

Diferentemente dos anteriores, hoje em dia não é necessário utilizar ";" no final da marcação no Javascript. mas EU vou utilizar, pois assim eu marco onde terminei um código e acho que fica mais legível.

dentro do "<script>XYZ</script>" escreveremos o código "window.alert('x')". 
É como o "print(x)" do Python. +/-. Aparece um alerta no topo central da página.
ex:
<script>
	window.alert('Minha primeira mensagem!'); 									#Não é necessário o "window." e ";", mas legibilidade é legal
</script>

Quando fazemos isso, salvamos e recarregamos nossa página, a mensagem aparece primeiro, e só DEPOIS que aparece o título e o parágrafo, mesmo que o script esteja no final. Mas a cor ta lá.

Resumão: HTML é o conteúdo, o CSS é a estilização do conteúdo, e o Javascript é a iteração, a programação.

Mais dois códigos:
