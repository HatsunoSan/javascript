Variáveis são lugares onde as informações do programa ficarão gravadas.
Assim como no Python, o sinal de igual ("=") significa "recebe"
ou seja: 
Variável_x = y
	Variável_x RECEBE y
Essas informações podem ser utilizadas ou requeridas pelo programa diversas vezes e utilizadas de diversas formas.

No Javascript Moderno, além de utilizar a palavra "var", também podemos usar a palavra "let".

Existe uma coisa chamada "string" nas Linguagens de Programação. São frases que ficam salvas na variável. 
Nós a salvamos na variável da seguinte forma:

var s1 = "Javascript"
var s2 = 'Curso em Vídeo'
var s3 = ´Wolly´

Sim, podemos salvar de três formas diferentes e as três formas diferem uma das outras, mas mais pra frente falaremos disso.

As variáveis tem nomes. O nome de uma variável é conhecido como IDENTIFICADOR.
	*Eles podem começar com LETRA, "$" ou "_"
	*NÃO podem começar com NÚMEROS.
	*É possível usar LETRAS ou NÚMEROS.
	*É possível usar ACENTOS e SÍMBOLOS.
	*NÃO podem conter ESPAÇOS
	*NÃO podem ser PALAVRAS RESERVADAS, como alert, prompt, confirm e etc

No Visual Studio Code, é possível abrir um NodeJS clica lá em cima na aba Terminal > New Terminal.
Com o Terminal aberto, escrevemos simplesmente "node" e pronto, node no VSCode
funciona igualzinho ao NodeJS instalado, mas dentro do VSCode.
Para sair do NodeJS, escrevemos ".exit", e para sair do Terminal do VSCode, digitamos "exit".
Se por acaso o NodeJS estiver cheio, CTRL = L para limpar.

IDENTIFICADORES DICAS:

	*MAIÚSCULAS e MINÚSCULAS fazem diferença.
	*Tente escolher NOMES COERENTES para as variáveis.
	*Evite se tornar um 'PROGRAMADOR ALFABETO' ou um 'PROGRAMADOR CONTADOR', aquele que chama as variaveis de n1, n2, n3 e etc
	
Variáveis servem para guardar dados!
Palavras, nomes, números, numeros negativos, frases. etc. 
O Javascript não difere um número de um número real, ou float, ou whatever.

5, 18, -12, 0.5, -15.9, 3.14, 8.0 - NÚMEROS. Ponto. Não diferem, sendo float ou não. Javascript os interpreta sendo do MESMO TIPO - O tipo number!

strings salvam frases ou palavras ou algo assim. 
CPF por exemplo, é uma string. Tem números? Tem, mas também tem outros símbolos, então apesar de ter números, não é numérico por conta dos simbolos.

ex de strings:
var empresa = "Google"
var linguagem = 'Javascript'
var nome = ´Wolly´
var cpf = 'xxx.xxx.xxx-xx'

True e False são do tipo Boolean, ou Boleano. São tipos primitivos Primordiais

	Infinity
